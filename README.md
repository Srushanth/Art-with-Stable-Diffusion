# Art-with-Stable-Diffusion

## Steps

- Create a new Virtual Environment.
```bash 
conda create -p .venv python==3.11 -y
```
- Create `requirements.txt`
    - pytest
    - mkdocs
    - mkdocs-material
- Create a new mkdocs project.
```bash
mkdocs new .
```
- Create `Makefile`
- Create necessary directories.
    - notebooks
    - src
    - tests
- Source control
    1. git status
    1. git add .
    1. git commit -m ""
    1. git push origin main
